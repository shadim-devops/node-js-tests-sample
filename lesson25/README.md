# Jenkins Basic Pipeline for Lesson25

## Description
This project demonstrates a basic Jenkins pipeline.

## Stages
1. Checkout – simulate downloading code
2. Build – simulate build process
3. Test – simulate testing
4. Deploy – simulate deployment

## Features
- simple pipeline with 4 stages
- sequential execution
- basic CI/CD workflow
