# Jenkins Advanced Pipeline

## Description
This project demonstrates a Jenkins pipeline with:
- error handling (try-catch)
- conditional stage execution
- email notifications
- environment variables

## Stages
1. Checkout – download code
2. Build – simulate build
3. Test – simulate testing
4. Deploy – simulate deployment

## Features
- try-catch in each stage
- conditional Deploy stage
- email notifications (success/failure)
- includes author, time, result
